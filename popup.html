Keyboard Navigation

Usage

  1. Press Ctrl+Space to highlight links
  2. Use the numbers to choose the link to follow
  3. (Use Backspace to correct, Escape to cancel)
  4. Press Enter to follow the link

Custom shortcut

Use the built-in Chrome shortcut editor:

chrome://extensions/configureCommands

You need to copy & paste this url into a new tab. Linking to a chrome:// URL doesn't work here, sorry.
